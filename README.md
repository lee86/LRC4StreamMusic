# qq音乐歌词接口

## 描述

此程序包含两个接口，分别用于返回`歌曲列表`和`歌词`

本意是针对群晖audio歌词插件进行精简后修改，主要是原php不会，所以分离成server + client形式，方便直接调用

流程相对于原插件多了一次中间查询返回的操作，个人自选

## 调用方式

http GET

http://127.0.0.1:8080/music?musicname=[查询歌曲、歌手]
